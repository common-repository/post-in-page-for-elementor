== Description ==

Show a simple post in an Elementor page

== Installation ==
* install [Elementor](https://be.elementor.com/visit/?bta=207645&brand=elementor)
* install the plugin

== How to use it ==

* add "Show post" widget in your page
* select post in drop-down
